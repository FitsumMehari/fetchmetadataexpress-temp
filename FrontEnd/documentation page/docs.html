How to Create an Account

To generate Metadata:

First, Go to Landing Page and click “Get Started”.

Second: Input your name, Email address and password

Third: Create account.

If you already have an account, all you have to do is Sign In.

How to Generate Metadata
  • Sign in and click on upload file
  • Drag and drop file or click on “Choose File” and select the file you want to upload
  • Click on “Generate Metadata”
  • Voila!, your uploaded file metadata is generated instantly.
What to do with Generated Metadata

After your Metada has been successfully generated, you can choose to do one of three things.You can either:

  1. Download as a file type (PDF, JPEG) to your device
  2. Share on the social media platform of your choice or on all of them
  3. Save for when you need it.
